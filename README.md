## 📋 프로젝트

### 1. 📝 텍스트 드롭 PRO

**텍스트 관리 + 엑셀 뷰어 통합 도구**

실무에서 자주 쓰는 텍스트를 빠르게 관리하고, 엑셀 파일을 간편하게 조회할 수 있는 올인원 도구입니다.

**주요 기능:**
- 📂 카테고리별 텍스트 관리
- 📋 원클릭 복사
- 🎯 드래그 앤 드롭
- 📊 엑셀 뷰어 (검색/필터)
- 💾 자동 저장 (로컬)
- 🔒 완전한 개인정보 보호

**[📖 상세 보기](./text-drop-pro/) | [⬇️ 다운로드](./text-drop-pro/text_drop_pro.html)**

---

### 2. 📄 Excel-PDF 변환기 (준비 중)

**엑셀 링크를 PDF로 대량 자동 변환**

구글 스프레드시트 링크를 자동으로 열어 PDF로 저장하는 프로그램입니다.

**주요 기능:**
- 📁 시트별 폴더 자동 생성
- 👤 구글 계정 다중 관리
- ⚡ 대량 변환 (100개 이상)
- 🔄 원본 파일명 유지
- 📊 실시간 진행상황

**곧 공개 예정!**

---

### 3. 📸 건설 사진 관리 앱 (준비 중)

**협력업체 공사 사진 관리 시스템**

Firebase 기반 PWA로 협력업체들이 공사 사진을 쉽게 업로드하고 관리할 수 있습니다.

**곧 공개 예정!**

---

### 4. 🏪 매장 관리 시스템 (준비 중)

**1,700개 매장 협력사 정보 관리**

매장별 협력업체 정보를 한눈에 확인하고 관리하는 시스템입니다.

**곧 공개 예정!**

## 🚀 시작하기

### 텍스트 드롭 PRO 사용법

**1단계: 다운로드**
```
text_drop_pro.html 파일 다운로드
```

**2단계: 실행**
```
파일 더블클릭 → 브라우저에서 자동 실행
```

**3단계: 사용**
```
✅ 텍스트 저장/관리
✅ 엑셀 파일 조회
✅ 클릭으로 복사
```

**끝!** 🎉

## ✨ 특징

### 📝 텍스트 관리
- 🏷️ **카테고리 관리** - 업무, 개인 등으로 분류
- 📋 **원클릭 복사** - 클릭 한 번으로 클립보드 복사
- 🎯 **드래그 앤 드롭** - 드래그해서 다른 곳에 붙여넣기
- ✏️ **수정 기능** - 저장된 텍스트 언제든 수정
- 🔍 **카테고리 필터** - 원하는 카테고리만 빠르게 확인

### 📊 엑셀 뷰어
- 📁 **파일 업로드** - 드래그 앤 드롭 또는 클릭으로 업로드
- 📑 **멀티 시트** - 여러 시트 탭으로 전환
- 🔎 **실시간 검색** - 전체 또는 컬럼별 검색
- 📋 **셀 복사** - 셀 클릭으로 즉시 복사
- 🎨 **하이라이트** - 검색어 자동 하이라이트
- 🔽 **컬럼 필터** - 특정 컬럼만 검색

### 🎨 공통 기능
- 💾 **자동 저장** - 브라우저 로컬에 자동 저장
- 🔒 **완전 프라이버시** - 모든 데이터 로컬 저장
- 📱 **반응형 디자인** - 모바일/태블릿 지원
- 🎯 **직관적 UI** - 사용하기 쉬운 인터페이스
- 🆓 **완전 무료** - 라이선스, 로그인 불필요

## 🚀 사용 방법

### 1단계: 다운로드
```
text_drop_pro.html 파일 다운로드
```

### 2단계: 실행
```
파일 더블클릭 → 브라우저에서 자동 실행
```

### 3단계: 텍스트 관리
```
✅ 카테고리 입력 (예: 업무, 개인)
✅ 라벨 입력 (예: 이메일 템플릿)
✅ 내용 입력
✅ 저장 버튼 클릭
✅ 카테고리별로 필터링
✅ 클릭으로 복사 또는 드래그로 붙여넣기
```

### 4단계: 엑셀 뷰어
```
✅ 엑셀 뷰어 탭 클릭
✅ 엑셀 파일 드래그 또는 업로드
✅ 시트 선택
✅ 검색/필터 사용
✅ 셀 클릭으로 복사
```

**끝! 설치 불필요!** 🎉

## 🔄 데이터 백업

### 자동 저장
```
모든 데이터는 자동으로 localStorage에 저장
→ 브라우저 캐시만 지우지 않으면 계속 유지
```

### 백업 방법
```
방법 1: HTML 파일 복사
→ 같은 브라우저면 데이터 유지

방법 2: 브라우저 프로필 백업
→ Chrome: 설정 → 프로필 백업

방법 3: 정기적으로 엑셀로 내보내기 (수동)
```

## 🐛 문제 해결

### Q: 데이터가 사라졌어요
**A:** 브라우저 캐시를 지우면 localStorage도 삭제됩니다.
```
해결: 정기적으로 중요 데이터 따로 백업
```

### Q: 엑셀 파일이 안 열려요
**A:** 지원하는 형식인지 확인하세요.
```
지원: .xlsx, .xls, .csv
미지원: .xlsm (매크로), .numbers (Mac)
```

### Q: 엑셀 파일이 깨져요
**A:** 파일 인코딩 문제일 수 있습니다.
```
해결: Excel에서 다시 저장 후 재업로드
```

### Q: 모바일에서도 되나요?
**A:** 네! 모바일 브라우저에서도 작동합니다.
```
단, 드래그 앤 드롭 대신 클릭으로 사용
```

### Q: 여러 PC에서 동기화되나요?
**A:** 아니요. 각 브라우저에 독립적으로 저장됩니다.
```
동기화 원하면: HTML 파일 복사 또는 클라우드 저장
```

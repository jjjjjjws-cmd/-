# Excel to PDF 변환 프로그램 v3.0 (무료 버전)

## 🎉 소개

Excel 파일에 있는 구글 스프레드시트 링크를 자동으로 PDF로 변환하는 프로그램입니다.

## ✨ 주요 기능

- ✅ **시트별 폴더 자동 생성**: 각 시트의 PDF를 별도 폴더에 저장
- ✅ **구글 계정 다중 관리**: 여러 구글 계정 등록하고 선택 사용
- ✅ **실시간 진행상황**: 처리 중인 항목과 통계를 실시간으로 표시
- ✅ **일시정지/재개**: 작업을 일시정지하고 재개할 수 있음
- ✅ **에러 로그**: 변환 실패 항목을 자동으로 기록
- ✅ **완료 통계**: 성공/실패 개수와 소요 시간 표시

## 📋 시스템 요구사항

- Windows 10/11
- Python 3.8 이상
- Google Chrome 브라우저
- 인터넷 연결

## 🚀 설치 방법

### 1. Python 설치

Python이 설치되어 있지 않다면:
1. [Python 공식 사이트](https://www.python.org/downloads/) 방문
2. 최신 버전 다운로드 및 설치
3. 설치 시 "Add Python to PATH" 체크 필수

### 2. 필요 패키지 설치

명령 프롬프트(cmd)를 열고 다음 명령 실행:

```bash
pip install -r requirements_free.txt
```

또는 개별 설치:

```bash
pip install openpyxl selenium psutil
```

### 3. Chrome Driver 설치

Selenium이 자동으로 Chrome Driver를 관리하므로 별도 설치 불필요합니다.

## 📖 사용 방법

### 1. 프로그램 실행

```bash
python excel_to_pdf_free.py
```

또는 파일을 더블클릭하여 실행

### 2. 구글 계정 추가

1. "계정 추가" 버튼 클릭
2. 구글 계정 이메일 입력 (예: user@gmail.com)
3. 계정 선택

**처음 사용 시**: 크롬 브라우저가 열리면 해당 구글 계정으로 로그인하세요.

### 3. Excel 파일 준비

Excel 파일 구조:
```
시트1: 1월
  A열: 구글 스프레드시트 링크
  https://docs.google.com/spreadsheets/d/...
  https://docs.google.com/spreadsheets/d/...
  
시트2: 2월
  A열: 구글 스프레드시트 링크
  https://docs.google.com/spreadsheets/d/...
  https://docs.google.com/spreadsheets/d/...
```

### 4. 변환 설정

1. **엑셀 파일 선택**: "파일 선택" 버튼으로 Excel 파일 선택
2. **URL 열 지정**: 링크가 있는 열 문자 입력 (기본: A)
3. **시트 선택**: 처리할 시트를 선택 (여러 개 선택 가능)
4. **저장 폴더 선택**: PDF를 저장할 폴더 선택

### 5. 변환 시작

1. "변환 시작" 버튼 클릭
2. 진행 상황 확인
3. 필요시 "일시정지" 또는 "중지" 가능

### 6. 결과 확인

변환 완료 후:
```
선택한_폴더/
  ├─ 시트1/
  │   ├─ row_2.pdf
  │   ├─ row_3.pdf
  │   └─ ...
  ├─ 시트2/
  │   ├─ row_2.pdf
  │   ├─ row_3.pdf
  │   └─ ...
  └─ failed_items.txt (실패 항목이 있는 경우)
```

## 🎯 사용 예시

### 예시 1: 월별 매출 보고서

**상황**: 
- Excel에 12개 시트 (1월~12월)
- 각 시트에 구글 스프레드시트 링크 100개

**작업**:
1. 구글 계정 선택
2. Excel 파일 선택
3. 12개 시트 모두 선택
4. 저장 폴더: `D:\매출보고서_2024`
5. "변환 시작"

**결과**:
```
D:\매출보고서_2024\
  ├─ 1월\
  │   └─ (100개 PDF)
  ├─ 2월\
  │   └─ (100개 PDF)
  └─ ...
```

### 예시 2: 협력업체 견적서

**상황**:
- Excel에 시트 1개
- 50개 협력업체 견적서 링크

**작업**:
1. 해당 시트만 선택
2. 변환 시작

**결과**:
```
저장폴더\
  └─ 견적서\
      └─ (50개 PDF)
```


## 🔧 문제 해결

### 크롬이 자동으로 안 닫힘

작업관리자에서 Chrome 프로세스 수동 종료

### PDF가 생성되지 않음

1. 구글 계정 로그인 확인
2. 스프레드시트 접근 권한 확인
3. 인터넷 연결 확인

### "모듈을 찾을 수 없음" 오류

```bash
pip install --upgrade openpyxl selenium psutil
```

### 변환이 너무 느림

- 링크당 약 5초 소요 (정상)
- 100개 링크 = 약 8~10분
- 네트워크 속도에 따라 달라질 수 있음

## 📊 성능

- **처리 속도**: 링크당 약 5초
- **안정성**: 에러 발생 시 자동 기록 후 계속 진행
- **메모리**: 약 200~300MB
- **CPU**: 크롬 실행 시 일시적으로 높아짐

## ⚠️ 주의사항

1. **대량 작업 시**: 100개 이상 링크는 시간이 오래 걸립니다
2. **권한 확인**: 스프레드시트 접근 권한이 있어야 합니다
3. **네트워크**: 인터넷 연결이 필요합니다
4. **크롬 프로세스**: 작업 중 크롬을 수동으로 닫지 마세요
